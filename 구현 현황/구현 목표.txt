===== DevQuest 구현 목표 =====

[최종 목표]
개발자 업무 챌린지 보상 시스템을 실제로 만들면서
동시성/정합성 문제를 AWS 환경에서 재현하고 해결한다.

[구현 목표 목록]

  1. concurrency-lab - 챌린지 + 포인트 기능 구현
     - 챌린지 CRUD + 완료 처리 + 포인트 지급
     - 포인트 잔액/이력 관리
     - payment-api 호출 클라이언트

  2. payment-api - 상품 + 주문 기능 구현
     - 상품 관리 (등록, 목록, 재고)
     - 주문 내역 관리

  3. 프론트엔드 대시보드
     - Vite + shadcn/ui + TanStack
     - 챌린지/포인트/상품 화면 + 동시성 테스트 시각화

  4. 동시성 문제 재현 및 해결 (1~4단계)
     - 중복 지급 / Lost Update / 재고 초과 / 분산 트랜잭션

  5. AWS 배포
     - EC2에 전체 시스템 배포 + 실환경 테스트
