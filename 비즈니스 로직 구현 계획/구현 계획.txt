===== DevQuest 비즈니스 로직 구현 계획 (총괄) =====

[서비스 개요]
개발자 챌린지 완료 시 외부 쇼핑몰에서 보상 상품이 즉시 주문/배송되는 시스템
서버간 통신에서 발생하는 동시성/정합성/데이터 동기화 문제를 재현하고 해결한다

[서버 구성]
  서버1: DevQuest (Spring Boot :8080) - 챌린지 + 완료 처리
  서버2: 쇼핑몰  (NestJS :3000)      - 상품 + 주문 + 배송
  서버3: 프론트  (Vite :5173)         - 대시보드

[단계 요약]
  1단계: 기본 기능 + 서버간 통신        ← 다음 작업
  2단계: 동시성 문제 재현 + 락으로 해결
  3단계: 분산 트랜잭션 + 데이터 동기화 + AWS 배포

  각 단계의 상세 계획은 해당 폴더 참조
