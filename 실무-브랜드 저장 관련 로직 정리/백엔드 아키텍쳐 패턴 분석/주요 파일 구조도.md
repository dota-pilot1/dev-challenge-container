
## 패턴 해석
- `Controller -> Service -> Mapper(Java) -> Mapper(XML)`의 전형적인 구조
- 외부 API는 `Service -> MiddlewareApiManager`로 통합
- 재현 프로젝트도 동일 패턴으로 시작하면 실무 코드 비교가 쉬움
