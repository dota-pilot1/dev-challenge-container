# 헤더 메뉴 출력 로직

## 1. 개요
헤더 메뉴(Header Menu)는 시스템의 최상위 네비게이션으로, 데이터베이스 상의 **Level 3 (prog_lvl=3)** 메뉴 항목들을 표시합니다.

## 2. 출력 로직 (Header.jsx)

### 데이터 소스
- `useFwTab` hook을 통해 `fwMenu` 데이터를 받아옵니다.
- `fwMenu`에는 백엔드에서 받아온 전체 메뉴 트리 중 최상위(Depth 0) 메뉴 리스트가 담겨 있습니다.

### 렌더링
- `fwMenu` 배열을 `map` 함수로 순회하며 `<ul>`, `<li>`, `<button>` 태그로 렌더링합니다.
- CSS 클래스 `header-1depth` 영역 내에 표시됩니다.

### 환경별 필터링
- `settingMenu` 함수를 통해 운영(Production) 및 QA 환경에서는 특정 ID 패턴(예: ID 뒷자리가 '1299')을 가진 메뉴를 화면에서 제외하는 로직이 포함되어 있습니다. 이는 개발 중이거나 테스트용 메뉴를 일반 사용자에게 노출하지 않기 위함입니다.

### 상호작용
- 메뉴 클릭 시 `handleActive(item.id)` 함수가 호출됩니다.
- 이는 전역 상태(Active Tab)를 변경하여, 해당 대메뉴에 연결된 하위 메뉴들이 **Left Menu(사이드바)**에 나타나도록 트리거합니다.

## 3. UI 구성
- 로고 우측에 위치하며, 가로로 나열된 텍스트 버튼 형태입니다.
- 선택된 메뉴(Active)는 `active` CSS 클래스가 적용되어 강조됩니다.
